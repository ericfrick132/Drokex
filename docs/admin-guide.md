# Drokex — Guía de Administración

## Acceso
- Ingrese a /superadmin/login con sus credenciales de Super Admin.
- Desde “Tenants” puede ver métricas y “Impersonar” para entrar a un subtenant.

## Aprobación/Rechazo de empresas
- En el panel del tenant, use “Empresas Pendientes”.
- Aprobación: Aprueba y habilita la empresa.
- Rechazo: Indique un motivo. Opción para desactivar.

## Contenido de Landing (CMS mínimo)
- En /superadmin/landing edite Título, Subtítulo y CTA.
- Persistencia local (demo); puede integrarse a backend más adelante.

## Usuarios
- “Usuarios” agrupa por tenant; despliegue para ver detalles.

## Impersonación
- Use el icono de “login” en Tenants para navegar autenticado al subdominio.
